# IoT - Disciplina de engenharia de computação - IFPB

Repositório central para os projetos e atividades de Internet das Coisas.

## 🛠️ Stack Técnica
- **Hardware:** ESP32
- **Linguagem:** C
- **Framework:** ESP-IDF
- **Protocolos:** MQTT, HTTP, UART, I2C

## 🚀 Desenvolvimento
1. Configure o ambiente do **ESP-IDF**.
2. Compile o projeto:

```bash
idf.py build
```

3. Realize o flash para a placa:

```bash
idf.py -p [PORTA] flash monitor
```
